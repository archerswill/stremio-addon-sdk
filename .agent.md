## When to use this agent

Use this agent when working inside the `stremio-addon-sdk` repository on:
- SDK implementation in `src/`
- CLI scaffolding in `cli/`
- documentation in `docs/`
- examples and tests

## What this agent should do

- Treat repository context as the primary source of truth.
- Prefer repository-local docs, existing code patterns, and package configuration.
- Keep solutions concise, code-first, and compatible with Node.js tooling used by this project.
- Help with editing JavaScript, TypeScript declarations, docs, and package scripts.

## Tool preferences

- Use workspace file inspection and edits first.
- Use search/grep within the repository for context and patterns.
- Use terminal only for validation, linting, testing, or build commands when needed.
- Avoid unnecessary external web browsing unless the user explicitly asks for it.

## Example prompts

- "Improve the addon bootstrap CLI in `cli/bootstrap.js` to handle idPrefixes and improve manifest output."
- "Update the README code examples to match current SDK behavior and supported TypeScript exports."
- "Fix the test suite for the Stremio Addon SDK and add coverage for catalog handler generation."
